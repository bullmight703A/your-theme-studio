# KIDazzle Academy WordPress Theme

A modern, vibrant WordPress theme for early childhood education centers.

## Installation

1. Download the `kidazzle` folder
2. Upload to `wp-content/themes/` in your WordPress installation
3. Activate the theme in **Appearance > Themes**
4. Copy the logo file `kidazzle-logo.png` to `assets/images/`

## Setup

### Menus
- Go to **Appearance > Menus**
- Create a menu and assign to "Primary Menu" location

### Customizer Settings
- Go to **Appearance > Customize**
- **Contact Information**: Phone, email, hours
- **Social Media Links**: Facebook, Instagram, Twitter, YouTube
- **Site Identity**: Upload custom logo

### Required Pages
Create these pages and assign the corresponding templates:
- Home (set as front page)
- About (use "About Page" template)
- Programs
- Locations
- Resources
- Contact (use "Contact Page" template)

## Features
- Responsive design
- Custom color scheme (Indigo + Yellow)
- Lucide icons
- Customizer integration
- SEO-friendly structure
