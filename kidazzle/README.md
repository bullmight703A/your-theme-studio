# KIDazzle Academy WordPress Theme

A modern, vibrant WordPress theme for KIDazzle Academy - exceptional early childhood education centers.

**Last Updated:** December 31, 2024 (v1.0.3)

## Theme Details

- **Theme Name:** KIDazzle Academy
- **Version:** 1.0.3

- **Requires:** WordPress 6.0+

## Features

- Responsive design optimized for all devices
- Custom page templates for Programs, Locations, Resources, About, and Contact
- Customizer settings for contact information and social media links
- Lucide icons integration
- Modern CSS with custom properties
- Mobile-friendly navigation

## Installation

### Via WP Pusher (Recommended)

1. Install WP Pusher on your WordPress site
2. Connect your GitHub account
3. Add this repository
4. Set the subdirectory to `kidazzle` (if theme is in a subfolder)
5. Install the theme

### Manual Installation

1. Download or clone this repository
2. Copy the `kidazzle` folder to `/wp-content/themes/`
3. Activate the theme in WordPress Admin → Appearance → Themes

## Page Templates

The theme includes several custom page templates:

- **Front Page** - Homepage with hero, value props, programs, testimonials, and CTA
- **Programs** - Display all age-appropriate programs
- **Locations** - Show all KIDazzle locations with maps and contact info
- **Resources** - Parent resources, events, and community links
- **About** - Company story and mission
- **Contact** - Contact form and information

## Customizer Options

Navigate to Appearance → Customize to configure:

- **Contact Information**
  - Phone number
  - Email address
  - Business hours
  
- **Social Media Links**
  - Facebook
  - Instagram
  - Twitter
  - YouTube
